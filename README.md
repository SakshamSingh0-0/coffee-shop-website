# MUG & BEAN ☕✨

A beautifully crafted, modern, and fully responsive minimalist coffee shop landing page. Designed with an elegant UI/UX layout to provide a seamless browsing experience for coffee lovers.

## 🎨 Features
* **Elegant UI/UX:** A clean, aesthetic, and minimalist design tailored for a premium cafe feel.
* **Fully Responsive:** Optimized for all screen sizes, including desktops, tablets, and mobile devices.
* **Interactive Shopping Cart:** Dynamic sliding cart toggle preview for an enhanced user experience.
* **Dedicated Sections:** * **Home:** Captivating hero section with an inviting call-to-action.
  * **About Us:** Brand story, founder details, and vision.
  * **Menu & Products:** Beautifully arranged coffee varieties and packaged beans with interactive hover effects.
  * **Reviews:** Clean testimonials section showcasing customer feedback.
  * **Blogs:** Engaging grid displaying the latest coffee-related articles.
  * **Contact & Login:** Fully styled forms for user inquiries and authentication.

---

## 🛠️ Tech Stack & Skills Used
* **UI/UX Design:** User-centric wireframing, minimalist design patterns, spatial composition, and aesthetic typography alignment.
* **Frontend Development:** HTML5, CSS3 (Flexbox, Grid, Custom Animations), JavaScript (ES6 for DOM manipulation and interactive states).
* **Icons & Fonts:** Font Awesome icons and integrated premium Google Fonts.
